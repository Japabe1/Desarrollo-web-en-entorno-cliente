(Ejercicio 9)

Desarrolla una pequeña aplicación en TypeScript que permita gestionar alumnos.

1. Define una interfaz Alumno con:

  • id
  • nombre
  • notas (array de números)

2. Crea una clase AlumnoApp que implemente la interfaz y que incluya:

  • Un método para calcular la nota media.
  • Un método que devuelva un resumen del alumno.

3. En el programa principal:

  • Crea un array de alumnos.
  • Añade al menos tres alumnos.
  • Muestra el resumen de cada alumno.
  • Calcula y muestra la media global de todos los alumnos.
